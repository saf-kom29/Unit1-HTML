Trip Research

I am going to be traveling to Paris, and since this is my first time travelling to France, I'm not exactly sure what to do! So I can figure out what to do, I'm going to be using Official Tourist Website so I know which museums, monuments, and local areas to travel to! Although, I'm obviously going to need a place to stay. For that, I'm going to be using TripAdvisor Hotels in Paris to find a place to stay during my time there. I'm going to stay there for about 3 weeks, so I'll need the weather forcast. I'm using The Weather Forcast in Paris to check how the weather is in Paris and when I'll be able to go there.

Getting There and Staying There

Flight & Hotel Booking

Now that I know what I'm going to do, I need to get tickets and plan how I'm getting there. For the flight, I'm using Expedia-Flights to Paris to book a flight to Paris. For the hotel and apartment spaces, I'm using the amazing website called booking.com hotels for Paris to find the place I'll be staying.

Transportation Info

I'm going to be travelling all over Paris--not just the Eiffel Tower. For transportation, I'm either going to be renting a car on booking.com car renting, or I'm going to be taking the metro. To see the metro schedule, I'm using the Paris Metro Schedule.
